Le classi in Javascript

Le classi in JavaScript sono una caratteristica introdotta nel linguaggio a partire da ECMAScript 2015 (conosciuto anche come ES6). Consentono di definire modelli per creare oggetti con proprietà e metodi simili.

Ecco alcuni concetti chiave:

  • Definizione di classe: Per definire una classe, si utilizza la parola chiave class, seguita dal nome della classe. Ad esempio:

    class Persona {

    class Persona {

    // Definizione della classe

    }

  • Costruttore: Il costruttore è un metodo speciale all'interno di una classe che viene chiamato automaticamente quando si crea una nuova istanza della classe. Viene definito utilizzando il metodo constructor. Ad esempio:

    class Persona {

    constructor(nome, cognome) {

    this.nome = nome;

    this.cognome = cognome;

    }

    }

  • Proprietà e metodi: All'interno della classe, è possibile definire proprietà e metodi. Le proprietà sono variabili associate all'istanza della classe, mentre i metodi sono funzioni associate all'istanza della classe. Ad esempio:

    class Persona {

    constructor(nome, cognome) {

    this.nome = nome;

    this.cognome = cognome;

    }


    saluta() {

    console.log(Ciao, sono ${this.nome} ${this.cognome});

    }

    }

  • Creazione di un'istanza: Per creare un nuovo oggetto basato su una classe, si utilizza la parola chiave new seguita dal nome della classe e eventuali argomenti richiesti dal costruttore. Ad esempio:

    const persona1 = new Persona('Mario', 'Rossi');

  • Ereditarietà: È possibile creare una gerarchia di classi in JavaScript utilizzando l'ereditarietà. Una classe figlia può estendere una classe genitore utilizzando la parola chiave extends. Ad esempio:

    class Studente extends Persona {

    constructor(nome, cognome, corso) {

    super(nome, cognome); // Chiamata al costruttore della classe genitore

    this.corso = corso;

    }


    studia() {

    console.log('${this.nome} sta studiando ${this.corso}');'

    }

    }
