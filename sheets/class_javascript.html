Le classi su Javascript

Le classi in JavaScript sono state introdotte con ECMAScript 6 (ES6) e forniscono una sintassi più pulita e leggibile per creare oggetti e gestire l'ereditarietà rispetto ai costruttori e ai prototipi tradizionali. Ecco le principali caratteristiche delle classi in JavaScript:

Definire una Classe

Le classi in JavaScript sono definite usando la parola chiave class. Una classe può avere un costruttore, metodi e getter/setter. Esempio:

class Persona {

constructor(nome, eta) {

this.nome = nome;

this.eta = eta;

}


saluta() {

console.log("Ciao, mi chiamo ${this.nome} e ho ${this.eta} anni");

}

}


const persona1 = new Persona ("Valerio", 30);

persona1.saluta(); // Output: Ciao, mi chiamo Valerio e ho 30 anni


Costruttore

Il metodo constructor è un metodo speciale per creare e inizializzare un oggetto creato con una classe. Viene chiamato automaticamente quando si crea una nuova istanza della classe. Esempio

class Libro {

constructor(titolo, autore) {

this.titolo = Harry Potter;

this.autore = JK Rowling;

}

}


Metodi

I metodi definiti all'interno di una classe non richiedono la parola chiave function. Esempio:

class Automobile {

constructor(marca, modello) {

this.marca = marca;

this.modello = modello;

}


descrizione () {

return "Automobile: ${this.marca} ${this.modello}";

}

}


const auto = new Automobile("Toyota", "Corolla");

console.log(auto.descrizione()); // Output: Automobile: Toyota Corolla


Ereditarietà

L'ereditarietà viene implementata usando la parola chiave extends. La parola chiave super() viene usata per chiamare il costruttore e i metodi della classe base. Esempio:

class Animale {

constructor(nome) {

this.nome = nome;

}


faSuono() {

console.log("Suono generico");

}

}


class Cane extends Animale {

constructor(nome, razza) {

super(nome); // Chiama il costruttore della classe base (Animale);

this.razza = razza;

}


faSuono() {

console.log("Suono generico");

}

}


class Cane extends Animale {

constructor(nome, razza) {

super(nome); Chiama il costruttore della classe base (Animale);

this.razza = razza;

}


faSuono() {

console.log("Bau Bau");

}

}


const cane = new Cane("Fido", "Labrador");

cane.faSuono; // Output: Bau Bau

console.log(cane.nome); // Output: Fido

console.log(cane.razza); // Output: Labrador


Metodi Statici

I metodi statici sono definiti utilizzando la parola chiave static e possono essere chiamati senza creare un'istanza della classe. Esempio:

class Matematica {

static somma(a, b) {

return a + b;

}

}


console.log(Matematica.somma(5, 3)); // Output: 8


Proprietà e Metodi Privati

Con l'introduzione della sintassi ECMAScript 2022, è possibile definire proprietà e metodi privati utilizzando il prefisso #. Esempio:

class ContoBancario {

#saldo = 0;


deposita(importo) {

this.#saldo += importo;

}


preleva(importo) {

if (importo <= this.#saldo -= importo;

} else {

console.log("Saldo insufficiente");

}


mostraSaldo() {

console.log("Saldo: ${this.#saldo"});

}

}


const conto = new ContoBancario();

conto.deposita(100);

conto.preleva(30);

conto.mostraSaldo(); // Output: Saldo: 70


Getters e Setters

Le classi possono avere getter e setter per ottenere e impostare il valore delle proprietà.

class Persona {

constructor(nome) {

this._nome = nome;

}


get nome() {

return this._nome;

}

}


const persona = new Persona("Mario);

console.log(persona.nome); // Output: Luca

persona.nome = "Andrea";

console.log(persona.nome); // output: Andrea


Espressioni di Classe

Le classi possono essere definite anche come espressioni. Esempio:

const Rettangolo = Class {
