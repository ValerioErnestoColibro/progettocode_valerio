Le classi di Javascript

Dato che un oggetto in JS è un dato con un valore, la proprietà è una chiave con cui scavare, e i metodi sono funzioni contestualizzate all'oggetto, perchè le stringhe hanno metodi e proprietà? Perchè in JS puoi trovare entrambi anche dentro una stringa segnata con "". Qui entrano in gioco le classi di JS, utili a creare oggetti (dato che JS è un linguaggio orientato agli oggetti). Tale oggetto viene considerato l'istanza della classe. Ecco un esempio:

class Auto {

constructor(marca, modello, anno, carburante) {

this.marca = marca; // Proprietà

this.modello = modello; // Proprietà

this.anno = anno; // Proprietà

this.carburante = carburante; // Proprietà

} }

La classe new Object permette di costruire un modello che crea per noi una struttura di dati. Esempio:

class Persona { constructor(nomeIniziale) { this.name = nomeIniziale; } }

Le classi di JS usano la lettera iniziale Maiuscola. Con la lettera iniziale minuscola, si tratta di una funzione.
