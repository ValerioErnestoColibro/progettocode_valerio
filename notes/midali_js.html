Introduzione a JS

  • JS è il linguaggio di programmazione che offre dinamicità alla pagina;
  • Possiamo salvare dei dati dentro delle variabili e molto altro;
  • Ci sono anche gli APA, che sono dei livelli aggiunti sul linguaggio JS;
  • L'interpretazione è dall'alto verso il basso;
  • JS è un codice interpreato, cioè viene tradotto dal browser;
  • Invece un codice compilato viene tradotto in linguaggio macchina e poi tradotto dal browser (C++);
  • JS è un linguaggio di programmazione lato client (ossia accade nel nostro browser), i codici lato server invece accadono solo parzialmente da noi. La maggior parte viene calcolato in un computer esterno (Java, Python);
  • Internamente si può implementare JS col tag 'script';


Le variabili su JS

  • La variabile è un gruppo di dati;
  • La variabile si dichiara con let nome = "testoVariabile" e per richiamarla si usa console.log(nome) e avremo come output testoVariabile;
  • Ricorda che nome, Nome e NOME non sono riconosciuti allo stesso modo da JS;
  • Al massimo puoi unire due parole così: nomeUtente;
  • Non iniziare i nomi delle variabili con numeri o segni di punteggiatura;
  • Per inizializzare la variabile: let nome = 'Valerio';
  • Si può sia usare var che let per manipolare le variabili. E' comunque più usato let rispetto a var QUESTO PER LE VARIABILI RIASSEGNABILI;
  • Si può usare anche const nome = "Mario" per dichiarare una variabile che non si riassegna. Ancora, si può usare const per valori che non cambiano come l'indirizzo di un database. Esempio: const persona = 'database:hosting.com';;
  • Per aggiornare una variabile dopo averla dichiarata, basta fare: nome = 'Ernesto';
  • Le variabili possono essere numeri interi o numeri con la virgola (float). Oppure se c'è del testo, si chiamano 'stringhe' in quanto segnati da semplici apici ''. Infine possono essere anche booleani, ossia ti danno come risultato 'true' o 'false';
  • L'array è una collezione di dati. Si apre con le []: let collezione = [1, 2, 3, 4, 5] oppure del testo come let listaPersone = ['Aldo', 'Giovanni', 'Giacomo'];
  • Gli array partono a contare la posizione degli elementi da 0, mentre l'indice (index) parte da 1;
  • Se voglio conoscere l'elemento in posizione 1, basta fare: console.log(listaPersone[1]); e avrò come output 'Giovanni';
  • Per oggetti si intendono degli schedari informatici di soggetti/oggetti reali. Queste informazioni sono date da proprietà, e le proprietà sono rappresentate da una coppia chiave-valore. Ogni proprietà di un oggetto può essere vista come una variabile che è collegata all'oggetto e che ha un nome (chiave) e un valore. Esempio:

    let persona = {

    nome: 'Mario',

    cognome: 'Rossi',

    eta: 30

    nazionalita: 'Italia',

    macchinePossedute: ['Panda', 'Polo'] // Dato che ci sono varie opzioni, si crea un array;//

    };

    Per richiamare una caratteristica, basta fare: console.log(persona.nome); e darà sulla console la stringa 'Mario';

    Se invece voglio conoscere una caratteristica dentro un array, farò: console.log(persona.macchinePossedute[1]) e avrò come output la stringa 'Polo';

Lavorare con i numeri

  • I numeri possono essere interi (10), float (1.1234567), double (1.1234567890), binary (0 e 1), octal ed esadecimali;
  • Questi dati offriranno sempre 'number' come output;
  • C'è un metodo chiamato 'toFixed' per visualizzare meglio i numeri dopo la virgola. Esempio: const numero = 1.23456789; se facciamo console.log(numero.toFixed(2)); avremo come output 1.23;
  • Si può anche convertire una stringa in un numero. Esempio: const numero = '43'; e poi faccio console.log(typeof numero); avrò come output la parola 'string';
  • Se invece vogliamo fare un'operazione (addizione, sottrazione, ecc.) con il "numero della stringa", basta fare: console.log(Number(numero) + 1); e l'output sarà 44;
  • Come operatori abbiamo: + - * / % ** (addizione, sottrazione, moltiplicazione, divisione, resto della divisione e potenza). Da qui si capisce che se il resto di una divisione è diverso da 0 allora è un numero dispari, se invece il resto è 0 avremo un numero pari;
  • La precedenza degli operatori: * e / hanno la precedenza su + e - ;
  • Operatori di incremento e decremento: let numero = 10; numero++; console.log(numero); avrà come output 11. Oppure si fa numero--; per avere come output 9;
  • Operatori di assegnazione: let numero = 10; numero =+ 5; console.log(numero) ci darà come output 15. Stesso discorso con -=, *= e /=;
  • Operatori di comparazione: == Esempio: let numero = 10; console.log(numero < 11); ci darà come output un valore booleano (in questo caso 'true'). Idem coi segni >, <= e =>. Poi ci sono anche == (per vedere se la variabile corrisponde al valore da noi indicato) oppure !== (per vedere se la variabile è diversa dal valore da noi indicato), e poi === per confrontare sia il valore che l'identità. Esempio: let numero = '10'; console.log(numero === 10) avremo come outpu 'false' perchè 10 è una stringa, non un numero;


Lavorare con le stringhe

  • Una stringa si crea così: let testo = "Questa è una stringa";
  • Si preferiscono "" a '' perchè col primo si possono inserire parole con l'apostrofo. Si può evitare usando il simbolo "escape\" con un esempio: let testo = 'L\'albero è verde';
  • Per concatenare le stringhe, basta fare: let nome = "Mario"; let testo = "Ciao mi chiamo" + nome; console.log(testo) ci darà come output "Ciao mi chiamo Mario";
  • Su JS, cose come let nome = 'Mario' + 56; console.log(nome) ci darà come output "Mario56". Per convertire da numero a stringa si usa il comando "toString", invece per andare da stringa a numero si usa "Number";
  • I "Template Literals" inseriscono dentro una stringa delle funzioni o delle variabili evitando l'uso di "escape\" con l'uso dei backtick ``. Esempio: let numero = `''L'albero l'ala l'uso''` oppure let testo = "Testo della variabile"; let elemento = `'h1'${testo} nuova'h1'` dopodichè lo mandiamo a schermo con document.body.innerHTML = elemento; e avremo il testo della variabile visualizzato dentro la stringa;


Metodi delle stringhe

  • Una stringa E' ANCHE UN OGGETTO e quindi ha anche dei metodi;
  • Per trovare la lunghezza di una stringa, basta fare: let strings = "Ciao"; console.log(string.length) ci darà come output 4 ossia gli elementi della stringa;
  • Per identificare un carattere specifico, facciamo: let strings = "Testo della stringa digitata"; console.log(strings[0]); ci darà come output "T" ossia il primo elemento della stringa. Invece per vedere l'ultima lettera si può fare: let stringa = "JavaScript"; let ultimaLettera = stringa[stringa.length - 1]; console.log(ultimaLettera); // Output: "t";
  • Prendere una parte di stringa: si fa console.log(strings.indexOf('stringa')); ci darà come output 11 ossia il punto di inizio della parola voluta. Si può anche fare let indexStringa = stringa.indexOf('stringa'); console.log(strings.slice(indexVariabile);) e avremo come output "digitata" perchè ha tagliato il resto;
  • Si può cambiare anche il "case" della stringa: console.log(strings.toUpperCase()) e darà come output "TESTO DELLA VARIABILE DIGITATA" scrivendo la stringa in maiuscolo. Invece con toLowerCase metteremo tutto minuscolo. Infine usiamo "replace" per sostituire una parola di una stringa: console.log(strings.replace('digitata', 'scritta')) e avremo come output "Testo della variabile scritta";


Array

  • Un array è un insieme di dati e si crea facendo let array = [1, 2, 3]; console.log(array) ci darà come output 1, 2, 3;
  • ANCHE L'ARRAY E' UN OGGETTO, dunque ci saranno dei metodi;
  • Anche qui si parte a contare da zero come per le stringhe;
  • Come per le stringhe, possiamo verificarne posizione dell'elemento, indice, lunghezza e simili con gli stessi metodi;
  • Per accedere ad un elemento specifico: console.log(array[numeroPosizioneElemento]);
  • Per convertire una variabile in un array, prendiamo queste due righe di codice: let citta = "Milano,Firenze,Roma,Napoli,Palermo"; let data = citta.split(','); console.log(data); avremo come output un nuovo array togliendo le virgole;
  • Si può anche fare il contrario, ossia convertire un array in una stringa. Esempio: let data = ["Milano", "Firenze", "Roma", "Napoli", "Palermo"]; let citta = data.join('-'); console.log(citta); e avremo una stringa di elementi testuali separate da un trattino;
  • Per aggiungere e rimuovere elementi da un array, si può fare così: data.push("Cagliari"); console.log(data); e avremo una nuova città in fondo al nostro array.
  • Per invece eliminare un elemento dell'array, basta fare: data.pop() e non avremo più "Cagliari" come elemento dell'array.
  • Per levare un elemento all'inizio dell'array, faremo: data.shift() e sparirà l'elemento "Milano";
  • Per invece aggiungere un elemento all'inizio dell'array, faremo: data.unshift("Torino"); e avremo l'elemento "Torino" all'inizio dell'array;

    if (nome == "Luca") {

  • Si può anche usare "Else if" nel caso in cui ci siano varie condizioni;
  • Nella comparazione, si usano == (uguale solo di valore), === (uguale valore e il tipo stringa, cifra, ecc.) e !== (diverso di valore e tipo);
  • Si possono anche mettere due o più condizioni dentro "If" usando &&:

    script

    if (numero < 20 && numero % 1) {

    }else{

    }

    script

Switch Statement

  • Se dobbiamo inserire troppi "If", si usa "Switch". Riprendendo gli esempi precedenti:

    script

    switch (nome) {

    case "Luca":

    console.log("Ciao Luca");

    break;

    case "Marco":

    console.log("Ciao Marco");

    break;

    case "Gino":

    console.log("Ciao Gino");

    break;

    default:

    console.log("Non conosco questo nome");

    }

    script

  • Il "break" serve per evitare che il codice vada avanti;
