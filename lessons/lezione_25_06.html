Lezione 25/06/2024

  • Cercare di non usare ChatGPT nella fase iniziale ossia QUANDO NON E' STRETTAMENTE NECESSARIO;
  • Differenze tra E-Commerce e MarketPlace:
    • E-Commerce = una sola persona vende;
    • MarketPlace = chiunque può vendere;
  • In un ipotetico MarketPlace, identifichiamo le entità e le proprietà:
    • Utenti:
      1. Nome utente;
      2. Password;
      3. Email;
      4. Indirizzo;
      5. Id utente;
    • Annunci:
      1. Data di creazione;
      2. Titolo;
      3. Descrizione
      4. Categoria;
      5. Status;
      6. Prezzo;
      7. Foto;
      8. Indirizzo;
      9. Id annuncio;
    • Recensioni:
      1. Nome utente recensore;
      2. Titolo;
      3. Descrizione;
      4. Votazione;
      5. Data di creazione;
      6. Id recensione;
  • L'organizzazione dei dati deve essere precisa perchè il cambiamento di un singolo elemento può rivoluzionare tutto il progetto;
  • Azioni dell'utente:
    1. Legenda: [ ] = funzioni di ogni proprietà;
    2. Registrarsi [token];
    3. Login [username, password];
    4. Logout [username, email, password];
    5. Modificare nome utente e password;
    6. Eliminare account;
    7. Segnalare(id, iduser, idads, descrizione, status ossia un booleano);
    8. Creare [], modificare[] o eliminare[] annuncio;
    9. Creare [], modificare[] o eliminare[] recensione;
    10. Contrassegna come "venduto";
    11. Lista filtrata (categoria, prezzo, ecc.);
    12. Crea, aggiungi o rimuovi Lista preferiti (id, iduser, idad);
    13. Crea, modifica o rimuovi lista recensioni utente;
