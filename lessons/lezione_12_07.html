Lezione 12/07/2024

  • Railway è un servizio online che ci affitta un server ONLINE per far girare la nostra applicazione;
  • Esporremo al pubblico le nostre API con un URL pubblico;
  • Fare corsi su AWS (altro servizio di cloud);
  • Puoi connetterti a Railway tramite l'account GitHub;
  • "Swagger" è una libreria che permette di documentare le API;
  • Fare ricerche su "Swagger";
  • YML è l'estensione dei file swagger;
  • Postman è un ambiente di prova IN LOCALE delle API;
  • Le API vanno connesse alle funzioni della tua app;
